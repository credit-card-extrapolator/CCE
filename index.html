Acerca de

Card Credit Extrapolator por sus siglas CCE, es una herramienta creada 100% con python, cuya función es extrapolar una CC(Card Credit) con los metodos mas reconocidos y funcionales de internet.

En la documentación oficial esta toda la información de lo que hace cada metodo y la manera como se puede hacer manualmente cada uno.

El codigo es muy sensillo puedes verlo en el repositorio de Git Hub de la pagina si lo deseas.

Version Web

Anteriormente CCE no estaba disponible para moviles, pero ahora todo cambio.

La version Web es lo mismo que la versión para pc, pero con la diferecia que esta no la tienes que descargar solo entrar a la web y listo, ademas es adaptable a pc y moviles.
Con una apariencia agradable a la vista y su facilidad y velocidad para utilizarlo lo hacen una buena herramienta.
Ademas de que cuando nos da el resultado tenemos dos botones uno para volver al inicio y otro para ir a la pagina generadora de tarjetas Namso Gen.
